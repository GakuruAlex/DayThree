# Leap Year #

A program that checks whether a given year is leap or not

## Usage ##

Run python main.py

### Tests ###

Run pip install -r requirements.txt
Run pytest -v

Examples:
    1904 True
    1992 True
    1996 True
    2148 True
    2196 True
    1900 False
    2100 False
    2200 False
    2300 False
    2500 False
    2700 False
